# LifeOS

LifeOS is an AI-powered personal life operating system (SaaS) to manage tasks, notes, goals, habits, schedules, expenses, and personal knowledge in one unified product.

This repository will be built in phases. This PR covers **PHASE 1 only**: product/technical architecture, folder structure, database design, UI design system, and API planning (no runtime app code yet).
